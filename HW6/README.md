## 粒子系统 介绍
### BALL COLLISION 不断碰壁以及撞击的球
- 系统设置在随机位置产生随机速度且半径为50的3颗球
- 球的颜色为随机
- 球在碰壁后会反弹
- 其中两颗球撞击后会改变运行方向，并且换颜色
- 两颗球在撞击的几帧中，会出现一闪一闪的效果，其实如果放慢观看，会发现那些一闪一闪的效果其实是球的颜色不断在换，且换得太快

## 希望能呈现的效果
- 每个粒子在每次撞击后会随机地加速或减速
